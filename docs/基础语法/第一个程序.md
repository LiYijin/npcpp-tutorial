# 第一个程序

让我们从经典的"Hello, World!"程序开始，了解C语言的基本结构。

## Hello World程序

```c
#include <stdio.h>

int main() {
    printf("Hello, World!\n");
    printf("欢迎来到C语言世界！\n");
    return 0;
}
```

## 程序解析

### 1. 预处理指令
```c
#include <stdio.h>
```
- `#include` 是预处理指令，用于包含头文件
- `stdio.h` 是标准输入输出库(Standard Input/Output Library)的头文件
- 这个头文件包含了 `printf()` 函数的声明

### 2. 主函数
```c
int main() {
    // 函数体
    return 0;
}
```
- `main()` 是C程序的入口函数，程序从这里开始执行
- `int` 表示函数返回整型值
- `return 0` 表示程序正常结束

### 3. 输出函数
```c
printf("Hello, World!\n");
```
- `printf()` 用于在控制台输出文本
- `\n` 是转义字符，表示换行
- 每个语句必须以分号(;)结束

## 编译和运行

### 1. 编写程序
将上述代码保存为 `hello.c` 文件。

### 2. 编译程序
在终端中执行：
```bash
gcc hello.c -o hello
```

### 3. 运行程序
```bash
./hello
```

### 4. 输出结果
```
Hello, World!
欢迎来到C语言世界！
```

## 更多示例

### 示例1：输出个人信息
```c
#include <stdio.h>

int main() {
    printf("姓名：张三\n");
    printf("年龄：20\n");
    printf("专业：计算机科学\n");
    printf("爱好：编程\n");
    return 0;
}
```

### 示例2：简单的计算
```c
#include <stdio.h>

int main() {
    printf("5 + 3 = %d\n", 5 + 3);
    printf("10 - 4 = %d\n", 10 - 4);
    printf("6 * 7 = %d\n", 6 * 7);
    printf("15 / 3 = %d\n", 15 / 3);
    return 0;
}
```

### 示例3：使用转义字符
```c
#include <stdio.h>

int main() {
    printf("引号的使用：\"Hello\"\n");
    printf("制表符：第一列\t第二列\n");
    printf("反斜杠：\\\n");
    printf("换行符的使用：\n这里是新的一行\n");
    return 0;
}
```

## 常用转义字符

| 转义字符 | 含义 |
|---------|------|
| `\n` | 换行 |
| `\t` | 水平制表符 |
| `\\` | 反斜杠 |
| `\"` | 双引号 |
| `\'` | 单引号 |
| `\r` | 回车 |
| `\b` | 退格 |

## 编译器选项

GCC编译器常用选项：

```bash
# 基本编译
gcc hello.c -o hello

# 显示所有警告信息
gcc -Wall hello.c -o hello

# 生成调试信息
gcc -g hello.c -o hello

# 优化代码
gcc -O2 hello.c -o hello

# 指定C标准
gcc -std=c99 hello.c -o hello
```

## 常见错误

### 1. 缺少分号
```c
// 错误
printf("Hello, World!\n")  // 缺少分号

// 正确
printf("Hello, World!\n");  // 添加分号
```

### 2. 缺少头文件
```c
// 错误 - 没有包含stdio.h
int main() {
    printf("Hello");  // 编译器不认识printf
    return 0;
}

// 正确
#include <stdio.h>
int main() {
    printf("Hello");
    return 0;
}
```

### 3. 函数拼写错误
```c
// 错误 - print拼写错误
print("Hello, World!\n");

// 正确
printf("Hello, World!\n");
```
