## 基本概念

### 程序结构
C语言程序的基本结构包括：
- 预处理指令（#include）
- 函数声明和定义
- 主函数 main()
- 语句和表达式

### 语法规则
- 每个语句以分号(;)结束
- 使用花括号({})组织代码块
- 区分大小写
- 注释可以使用 // 或 /* */
