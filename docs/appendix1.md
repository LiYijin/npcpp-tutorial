### 附录B：附属功能

6.2.1 control功能

@control注解可以应用于函数，显示地指示此函数是控制流处理的一部分。

```cpp
示例：

@control void my_control_func()
{
}
```

6.2.2 parser功能

@parser注解可以应用于函数，以便为编译器指定此函数是parser状态处理的一部分。

```cpp
示例：
@parser void parse_IPv4()
{
}
```

6.2.3 版本号注解

@app_version注释可以用于设置版本号，编译器在生成bin时将版本号信息写入，用于版本升级等场景。

```cpp
示例：
@app_version("src_mac_filter_v1.0");
```
